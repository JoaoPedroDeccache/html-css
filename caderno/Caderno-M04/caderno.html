Cap-24 - Formulários

Aula 01/ex-018 - como criar Formulários

Para criar um formuláeio utilizasse a tag <form> </form>

Para características de texto como nome e sobrenome é necessário adicionar a tag <input> type=text. Isso fará com que se crie uma caixa de resposta para ser preenchida com as informações pedidas

Para criar um botão de enviar é necessário adicionar a tag input com type=submit e value=enviar

Aula 02/ex-018 - utilização de label

É importante a utilização de label para fazer uma ligação entre o dado pedido e o campo que vai ser preenchido no formulário

para isso é só envelopar a palavra antes do campo que deve ser preenchido com a tag <label>

Aula 03/ex-018 - GET e POST

Método get é usado para dados não sensiveis, como: Nome , Sobrenome, Idade, Altura, Peso, etc...

O método get só suporta até 3000 bytes

Método post é usado para dados sigilosos, como: senhas, numero de cartão, imagens, etc...

O método post não é seguro para esconder os dados, ele apenas tira os dados da url. Para esconder os dados é necessário estudar https e mais para saber sobre essa àrea de criptografia de informações

Aula 04/ex-018 - caixas de texto e de senha

Para desligar o preenchimento automatico é preciso adicionar dentro da tag form (autocomplete=off)

Colocar o autocomplete dentro da tag input dará muito mais opções para informar doque aquele campo se trata

para usar o autocomplete dentro da tag input é preciso estar com o autocomplete de dentro da tag form ligado

dentro da tag form o (action) é para onde os dados do formulário serão mandados

A tag <input> type=reset servirá para limpar os campos do formulário

Adicionar required dentro da tag <input> tornará o campo ser de preenchimento obrigatório

Adicionar minlength e maxlength dentro da tag <input>definirá a quantidade mínia e máxima de carácteres no campo

Em campos de senhas há 2 autocomplete possiveis: current-password e new-password

current: para senhas já gravadas no navegador, usado em telas de login por exemplo

new: para senhas sendo criadas, em telas de criação de contas por exemplo

Adicionar placeholder dentro da tag <input> para colocar uma dica doque deve ser escrito no campo

Aule 05/ ex-018 - number, month, date e time

Também é possivel adicionar nos formulários campos de datas, números, horas, entre outros

Para fazer isso só é preciso alterar o "type" da tag <input>. EX: type=(number, date, month, time)

Para as caixas de números são utilizados os elementos min e max caso queira definir uma quantidade miníma ou maxima aos números. EX: min=0 max=10 (qualquer númeo acima de 10 não será aceito)

O elemento value é utilizado para definir um valor padrão ao abrir um formulário. EX: value=5 (o 5 já estará preenchido no campo ao inves de estar vazio)

O elemento step server para definir qual será o intervalo de números aceito. EX: step=0.5 (1 - 1,5 - 2 - 2,5 - 3). Se o step for de 0.1 por exemplo aceitará todas as dezenas (1,1 - 1,2 - 1,3 - 1,4 - 1,5 - 1,6)

Aula 07/ex-018 - Telefone, email e fieldset

Para adicionar campos para email e telefone é preciso modificar o type da tag <input> para type=email ou type=tel

Também é possivel agrupar varios campos juntos com a tag <fieldset> todos os parágrafos que desejam ser agrupados precisam estar dentro da tag

e com a tag <legend> dentro da tag fieldset é possivel adicionar uma legenda ao grupo

Aula 08/ex-018 - chackbox e radio

Aula 09/ex-018 - color, range e file

elementos de formulário para cores, barra de opnião e fotos

Para cores type=color

Para barra de opnião type=range

Para imagens type=file

O elemento de imagens só funcionará com method:post

Aula 10/ex-018 - select, datalist e textarea

É possivel adicionar em formulários os elementos de listas de seleção e areas de texto

Para criar uma lista sem opção de escrita é necessário adicionar a tag <select> com a tag <option> dentro dela com cada opção a ser selecionada possivel

Para adicionar uma lista com opção de escrita é necessário adicionar a tag <input> com type=text e o elemento list, e embaixo deste input a tag <datalist> contendo no 'id' o mesmo nome dado ao elemento 'list', dentro da datalist colocará as tags <option> com as opções disponiveis a serem escolhidas

Para adicionar uma area de texto adicionar a tag <textarea>
