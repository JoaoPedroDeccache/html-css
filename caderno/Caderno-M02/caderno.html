Efeito Gradiente

Na parte (head), cria um estilo de CSS para o (body) com as chaves ({}) e adiciona a tag <background-image:linear-gradient >. Dentro da tag definir primeiro a direção do degrade (to right, to top, to left,to bottom), podendo botar valores em graus para definir degrades em quaisquer direções incluido diagonais, e depois inserir as cores presentes no degrade separando-as com virgula.

Para definia o tamanho da cor na tela colocar a porcentagem de espaço da tela q ela ocupara antes da virgula. (EX: white 25%, black)

Fonte

Tipo de fonte

Para alterar o tipo de fonte do site utilizar a tag <font-family>

Fontes serifadas como times new roman dificultam a leitura em telas e são mais recomendadas para o papel

Fontes sem serifa como Arial confundem a leitura no papel e são recomandadas para a leitura em telas

Tamanho da fonte

Para alterar o tamanho da fonte utilizar a tag <font-size>

Medidas absolutas
cm, mm, in(polegadas), px, pt, pc

Medidas relativas
em, ex, rem, vw, vh, %

As medidas recomendadas para utilizar em font-size são em e px

1 em são 16 px que é o font-size padrão da maiorias dos navegadores

Peso, estilo

Para alterar o peso da fonte utilizar a tag<font-weight>

Existem os pesos lineares:lighter, normal ,bold e bolder

E também existem os pesos numéricos que vão de 100 a 900

Para alterar o estilo da fonte utilizar a tag <font-style>

Existem as variações italic e normal

Short hand font

Para diminuir todas as configurações de fonte em apenas uma linha

A ordem das configurações são:
font-style -> font-weight -> font-size -> font-family

Utilizar a tag <font>

Descobrir fontes de sites

Para descobrir as fontes usadas em determinado site utilizar a extenção fonts ninja instalada no crhome

Cap-15

Aula 01/ex 012 - Usando o id

Seletores

Para aplicar uma configuração de CSS para apenas um elemento utilizar a tag <id> e dar um nome (sem numeros) para identifica-lo

importante!

Tudo que em HTML é id, em CSS é #

Tudo que em HTML é class, em CSS é .

Aula 02/ex 012 - Usando class

Class é usado com a tag <class> para aplicar uma cofiguração para varios elementos, diferente do id que é usado para aplicar uma configuração para apenas um elemento

Aula 03/ex 013 - pseudo-classes

height= altura
width= largura

Pseudo-classes são representadas por dois pontos (:)

<div> é a tag para um texto ocupar toda a linha em que é escrito

usando a pseudo-classe hover ao passar o mouse em cima do texto o texto mudará sua forma, como cor, fonte entre outros. Poderá também mostra texto escondidos

Aula 04/ex 013 - pseudo-elementos

Pseudo-elementos são representados por dois pontos duas vezes (::)

Servem para botar elementos antes ou depois dos texto sem alterar no HTML

EX: a::after, a::before

após o comando mostrado acima utilizar a tag <content> para escrever o conteudo que vai estar antes ou depois destes textos

Resumo

# = id
. = class
: = pseudo-class
:: = pseudo-elementos
> = children

Aula 01 - Modelo de Caixas

Termos de medição das caixas

height = altura
width = largura
border = borda
padding = espaço entre o conteudo e a borda
margin = espaço fora da borda
outline = contorno fora da borda

Tipos de caixa

Box level

só começa em novas linhas e ocupa todo o espaço de tela disponivel

Inline level

não provoca quebras de linha e só ocupa o espaço de tela necessario para o conteudo
