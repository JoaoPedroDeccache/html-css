Aula-02/ex-015 - Colocando imagem de fundo

Para adicionar uma imagem de fundo é possivel usar 3 maneiras:

background-color
background-image com um linear-gradient
background-image com url e adicionar uma imagem de plano de fundo

Aula-03/ex-015 - Imagens que se repetem no fundo do site

Para definir a largura e a altura da imagem de fundo é preciso adicionar a tag <background-size> com os valores de width e height desejados

Usando a tag <background-repeat> é possivel alterar os modos de repetição.Que são:
repeat = padrão
no-repeat = a imagem não será repetida
repeat-x = a imagem repetirá no eixo x
repeat-y = a imagem repetirá no eixo y

Aula-04/ex-015 - Posicionamento da imagem no fundo do site

Para alterar o posicionamento das imagens no fundo do site é necessário adicionar a tag <background-position>

Lembra que nesta tag primeiro adiciona a coluna vertical (left, center ou right) e depois a coluca horizontal (top, center ou bottom)

Lembrar de colocar: height: 97vh.Em html sem conteudo, somente com a imagem de fundo o body irá ficar pequeno e a imagem de fundo desconfigurado.

Aula-05/ex-015 - Mudando o tammanho da imagem de fundo do site

background-size: auto = irá seguir o automatico do navegador
background-size: contain = irá ocupar todo o epaço da tela mas irá deformar a imagem
background-size: cover = irá ocupar todo o espaço da tela sem deformar mas não exibirá a imagem inteira.

Prestar atenção no tamanho da view port (height), no background-position e no background-repeat

Aula-06/ex-015 - background attachment e short hand

background-attachment

Para evitar imagens se repetindo é necessário utilizar a tag <background-attachment>.
A variavel scroll é utilizada para fazer a imagem do fundo rolar junto com o conteudo da página.
A variavel fixed é utilizada para deixar a imagem de fundo fixa e a rolagem acontece apenas no conteudo da página

short hand

Na short hand background é necessário seguir a ordem: background-color > background-image > background-position > background-repeat > background-size > background-attachment

Na short hand a tag background-size não está funcionando no momento, mas está na regra da w3c.Se ainda não estiver funcionando adicione a short hand sem o background-size e coloque ele separado

Aula-07/ex-015 - Centralização Vertical

Para centralizar uma caixa verticalmente não será utilizando a margin igual utiliza-se para centralizar horizontalmente

Primeiramente é necessário adicionar duas caixas, a de fora vai ser o container e a que estiver dentro do container vai ser o conteudo.As caixas devem estar identificadas com id's

O container deve conter a tag <position>com a variavel relative.O conteudo deve conter a mesma tag mas com a variavel absolute

Com a poosição do conteudo absoluta é possivel adicionar as tag's <left> e <top>, nessas tag's é preciso adicionar valores de 50% nos dois

E por ultimo deve-se adicionar a tag <transform> com a variavel translate com os valores de (-50%, -50%)

Esta centralização só vai funcionar se a altura (height) do container estiver na escala vh (view port)
